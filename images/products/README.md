## Lưu ý:
- Định dạng ảnh: JPG, PNG, WebP
- Kích thước khuyến nghị: 800x800px hoặc lớn hơn (tỷ lệ 1:1)
- Nếu không có ảnh, website sẽ tự động hiển thị emoji thay thế
- Ảnh sẽ được tự động resize để phù hợp với giao diện

## Cách thêm ảnh:
1. Đổi tên file ảnh theo danh sách trên
2. Copy file vào thư mục `images/products/`
3. Làm mới trang web để xem ảnh
