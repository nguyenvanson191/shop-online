# HƯỚNG DẪN THÊM ẢNH SẢN PHẨM

## Cách 1: Kéo thả ảnh (Dễ nhất)

1. **Mở thư mục ảnh:**
   - Vào thư mục: `images/products/`
   - Bạn có thể mở bằng cách click chuột phải vào thư mục và chọn "Open in File Explorer" (Windows)

2. **Đổi tên ảnh:**
   - Đổi tên file ảnh thành tên đúng (ví dụ: `cardigan-navy.jpg`)
   - **QUAN TRỌNG:** Phải đúng tên file như sau:
     - `cardigan-navy.jpg` - cho Áo Cardigan Xanh Navy
     - `cardigan-red.jpg` - cho Áo Cardigan Đỏ
     - `lacoste-tote-teal.jpg` - cho Túi Lacoste Teal
     - ... (xem danh sách đầy đủ bên dưới)

3. **Copy ảnh vào thư mục:**
   - Copy file ảnh đã đổi tên
   - Paste vào thư mục `images/products/`

4. **Làm mới trang web:**
   - Mở `index.html` trong trình duyệt
   - Nhấn F5 hoặc Ctrl+F5 để làm mới trang
   - Ảnh sẽ tự động hiển thị!

## Ví dụ cụ thể:

### Ví dụ: Thêm ảnh Áo Cardigan Xanh Navy

1. Bạn có file ảnh: `anh-cardigan.jpg` (hoặc tên bất kỳ)
2. Đổi tên thành: `cardigan-navy.jpg`
3. Copy vào: `images/products/cardigan-navy.jpg`
4. Mở lại trang web → Ảnh sẽ hiển thị!

## Lưu ý quan trọng:

✅ **Định dạng ảnh:** JPG, PNG, WebP đều được
✅ **Kích thước:** Khuyến nghị 800x800px hoặc lớn hơn (tỷ lệ vuông 1:1)
✅ **Tên file:** PHẢI ĐÚNG CHÍNH XÁC (không có khoảng trắng, viết thường)
✅ **Vị trí:** PHẢI đặt trong thư mục `images/products/`

❌ Nếu tên file SAI → Ảnh sẽ không hiển thị, website sẽ dùng emoji thay thế
❌ Nếu đặt SAI thư mục → Ảnh sẽ không hiển thị

## Kiểm tra ảnh đã được thêm chưa:

Sau khi thêm ảnh, bạn có thể kiểm tra:
- Mở File Explorer
- Vào `images/products/`
- Xem file ảnh đã có trong đó chưa

## Cần giúp đỡ?

Nếu ảnh không hiển thị, kiểm tra:
1. Tên file có đúng không?
2. File có trong thư mục `images/products/` không?
3. Đã làm mới trang web (F5) chưa?
4. Định dạng file có đúng không? (.jpg, .png, .webp)
